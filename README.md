# react-native-wifi-manager

React Native WiFi manager for iOS and Android

## Installation

```sh
npm install react-native-wifi-manager
```

## Usage

```js
import WifiManager from "react-native-wifi-manager";

// ...

const result = await WifiManager.multiply(3, 7);
```
